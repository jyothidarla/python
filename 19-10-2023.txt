=============================
Following are the decision making statements in pyton.Decision making is also
known as conditional statements.Following are the decision making statements in 
python

->if statement

->if-else statement
->if-elif-else statement

Nested if statements:
=====================
Python supports nested if statements.It is also known a s Inner if statement.Nested
if statement or Nesested if statement can be represent as defining or decalring a if
statement inside another if statements.


if(cond): ---->Outer if statement
    ---
    if(cond): ---->Inner if statement
      ----

or 
if  cond:
     ----
     if cond:
         ---
